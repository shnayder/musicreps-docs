# Recommendation Strings Inventory

Current user-facing recommendation text patterns across the app.

## In-Skill Suggestion Cards (Practice Tab)

Generated by `buildRecommendationLines()` in `mode-ui-state.ts`. Shown as
structured `SuggestionLines` in the PracticeConfig "Suggested" view.

| Priority | Verb | Level format | Example |
|----------|------|-------------|---------|
| 1 | Review | `{longLabel}, ...` | "Review Frets 0–3, naturals" |
| 2 | Practice | `{longLabel}, ...` | "Practice 1–2 semitones apart" |
| 3 | Start | `{longLabel}` | "Start Frets 4–8, naturals" |
| 4 | Keep practicing | `{longLabel}, ...` | "Keep practicing Major keys (0–3 ♯/♭)" |

### Flat text (buildRecommendationText)

Used internally for the recommendation rationale string (passed to
`computePracticeSummary`). Joins lines with commas, lowercased verbs. Note:
automate recs map to literal `"automate"` — e.g., `"review Frets 0–3, practice
1–2 semitones apart, automate Major keys (0–3 ♯/♭)"`.

### Status labels

Computed by `statusLabelFromLevel()` in `mode-ui-state.ts`. Not shown. 

In
`PracticeCard` (the idle-state card in `mode-screen.tsx`) as
`practice-status-label` + `practice-status-detail`, but practice card is never shown to the user today.

| Speed | Label |
|-------|-------|
| >= 0.9 | Automatic |
| >= 0.7 | Solid |
| >= 0.3 | Learning |
| < 0.3 | Hesitant |

### Status detail

- Not started: `"{total} {itemNoun} to learn"`
- Started: `"{automatic}/{total} {itemNoun} automatic"`

## Home Screen Skill Cards

Generated by `computeSkillRecommendation()` in `home-recommendations.ts`. Shows
as `cueLabel` (bold) + `detail` on each skill card. Uses `longLabel` for group
labels.

| Type | cueLabel | Detail pattern | Example |
|------|----------|---------------|---------|
| review | Review | `"Review {labels}"` | "Review Frets 0–3, naturals" |
| keep-practicing | Keep practicing | `"Keep practicing {labels}"` | "Keep practicing 1–2 semitones apart" |
| learn-next | Learn next level | `"Learn {label}"` or `"Get started"` | "Learn Frets 4–8, naturals" |
| automate | Almost there | `"Almost there — {labels}"` | "Almost there — Major keys (0–3 ♯/♭)" |
| not-started | *(empty)* | *(empty)* | *(no display)* |
| automatic | *(empty)* | *(empty)* | *(no display)* |

Single-group modes omit the group label (e.g., just "Review" or "Keep
practicing").

## Group labels by mode

Recommendations use `longLabel` (falling back to `label`). These are the current
`longLabel` values:

| Mode | Groups (longLabel) |
|------|-------------------|
| Guitar Fretboard | Frets 0–3, naturals · Frets 0–3 ♯/♭ · Frets 4–8, naturals · Frets 4–8 ♯/♭ · Frets 9–12, naturals · Frets 9–12 ♯/♭ |
| Ukulele Fretboard | *(same pattern, 6 fret-range groups)* |
| Semitone Math | 1–2 semitones apart · 3–4 semitones apart · 5–6 semitones apart · 7–8 semitones apart · 9–11 semitones apart |
| Interval Math | Seconds (m2, M2) · Thirds (m3, M3) · Fourth & tritone (P4, TT) · Fifth & minor 6th (P5, m6) · Sixths & sevenths (M6, m7, M7) |
| Key Signatures | Major keys (0–3 ♯/♭) · Major keys (4+ ♯/♭) · Minor keys (0–3 ♯/♭) · Minor keys (4+ ♯/♭) |
| Scale Degrees | 4th & 5th degrees · 3rd & 7th degrees · 2nd & 6th degrees |
| Diatonic Chords | Primary chords (I, IV, V) · Secondary chords (ii, vi) · Tertiary chords (iii, vii°) |
| Chord Spelling | Major triads · Minor triads · Dominant 7th · Major 7th · Minor 7th · Diminished, augmented & half-dim · Suspended & 6th chords |

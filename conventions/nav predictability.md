## Convention

Navigation should be predictable. No difference in where links take you on
first vs later uses.

## Applies to

All navigation elements: skill cards, tab links, menu items.

## Rationale

User testing showed confusion when first tap went to info/about and later
taps went to practice. Use first-time banners or similar to guide users to
onboarding-relevant content instead of changing navigation targets.

## Examples

**Good:** Skill card always opens practice tab. First-time banner says
"New here? Check the info tab to learn what this skill is about."

**Bad:** First tap on skill card opens info tab; subsequent taps open
practice tab.

# Recommendation Strings Inventory

Current user-facing recommendation text patterns across the app.

## In-Skill Suggestion Cards (Practice Tab)

Generated by `buildRecommendationLines()` in `mode-ui-state.ts`. Shown as
structured lines in the PracticeConfig "Suggested" view and as flat text in
PracticeCard's recommendation box. 
❓what's a PracticeCard ?
| Priority | Verb | Level format | Example |
|----------|------|-------------|---------|
| 1 | Review | `{groupLabel}, ...` | "Review E string, A string" |
| 2 | Practice | `{groupLabel}, ...` | "Practice D string" |
| 3 | Start | `{groupLabel} — N new item(s)` | "Start G string — 8 new items" |
| 4 | Keep practicing | `{groupLabel}, ...` | "Keep practicing B string" |

Flat text joins all lines: `"review E string, practice D string, start G string
— 8 new items"`.

## Home Screen Skill Cards

Generated by `computeSkillRecommendation()` in `home-recommendations.ts`. Shows
as `cueLabel` (bold) + `detail` on each skill card.

| Type | cueLabel | Detail pattern | Example |
|------|----------|---------------|---------|
| review | Review | `"Review {labels}"` | "Review E string, D string" |
| keep-practicing | Keep practicing | `"Keep practicing {labels}"` | "Keep practicing G string" |
| learn-next | Learn next level | `"Learn {label}"` or `"Get started"` | "Learn B string" |
| automate | Almost there | `"Almost there — {labels}"` | "Almost there — E string" |
| not-started | *(empty)* | *(empty)* | *(no display)* |
| automatic | *(empty)* | *(empty)* | *(no display)* |

Single-group modes omit the group label (e.g., just "Review" or "Keep
practicing").

# Speed check per input type

Response times depend on context and button layout. Need separate speed
check calibration for different input types:
- 3x4 grid vs 6x2
- With vs without accidentals
- Fretboard taps

Adding accidentals to fretboard practice makes you much slower — more
motor than cognitive. Speed check should account for this.

# "All skills" hint text should be actionable

Hint text on the all-skills page should tell the user what to do, like on the
active tab ("Pick what you want to automate").

## Current behavior

The All Skills tab shows a hint at the top:

> Tap the ☆ on a skill to add it to your **Active** list.

This is mechanical/instructional — it describes *how* the star button works, not
*why* the user should care or *what* they're choosing.

Active skills tabs says

> Pick the skills you want to drill. Star them in **All Skills** to build your lineup.

## Proposed

Replace with actionable, goal-oriented text that motivates the decision:

> Star the skills you want to automate.

Short, action-oriented, parallel to the Active tab's framing. The star affordance
is self-explanatory from the UI — the hint should frame the *intent*, not the
*mechanism*. Use automate to reinforce brand. 

Also tweak active skills text:

> Star the skills you want to automate in **All Skills**.

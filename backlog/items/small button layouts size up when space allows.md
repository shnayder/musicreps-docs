# Small-button layouts: size up when space allows

Fretboard mode: 4+3 button layouts are smaller than 6+6 layouts — they should
be the same size. More generally, 4×3 and other small-count button layouts
appear too small across modes. When there's available space, buttons should
grow to use it, making them easier to tap.

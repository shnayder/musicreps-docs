# Pick sharps or flats at random each round

For fretboard memorization (and possibly other skills), pick sharp or flat
spelling at random each round. Otherwise the user may never see "Bb" because
it's always shown as "A#".

Note: this is a **display-only** change. The underlying items don't change —
only how each prompt is rendered.
